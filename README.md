# Deep Learning Course Materials

This repository contains a comprehensive set of Google Colab notebooks and resources for learning Deep Learning, from foundational concepts to advanced frameworks.

## ðŸ§  Neural Network Playground

Check out the interactive [Neural Network Playground](https://dlmastery.github.io/neural-network-playground/) to visualize how neural networks learn.

### Key Features of the Playground:
- **CNN Explainer**: Visualize how Convolutional Neural Networks process images, including layers, activations, and feature maps.
- **Transformer Explainer**: Understand the self-attention mechanism, tokenization, and how Transformers generate text.
- **Diffusion Model Explainer**: Learn how diffusion models generate images by reversing the noise process.
